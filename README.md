Шаги воспроизведения:
1. Ввести в поле email «user@example.com»
2. Нажать кнопку «Проверить»
Ожидаемый результат: Появился alert с текстом «Прекрасный email-адрес»

*) Невалидная структура email

Название: Отсутствие символа @
Шаги воспроизведения:
1. Ввести «plainaddress»
2. Нажать кнопку «Проверить»
Ожидаемый результат: Под кнопкой появилась ошибка красного цвета: «Введен некорректный email»

Название: Отсутствие локальной части (до @)

*) Поведение формы

Название: Ввод email без нажатия кнопки
Шаги воспроизведения:
1. Ввести «invalid-email»
2. Не нажимать кнопку «Проверить»
Ожидаемый результат: Никаких сообщений не появляется

Название: Повторная проверка после исправления
Шаги воспроизведения:
1. Ввести «invalid», нажать «Проверить» → появилась ошибка
2. Исправить на «valid@example.com», нажать «Проверить»
Ожидаемый результат: Ошибка исчезла, появился alert: «Прекрасный email-адрес»

Название: Пустое поле
Шаги воспроизведения:
1. Оставить поле пустым
2. Нажать кнопку «Проверить»
Ожидаемый результат: Под кнопкой появилась ошибка красного цвета: «Введен некорректный email»
